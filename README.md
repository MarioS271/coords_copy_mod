## Minecraft Mod to copy coordinates of the block you're currently looking at with the press of a button.
### The Mod is fabric 1.20.2 and I will try to always have it up to date.
<hr>
Current Features (v1.0):
- Copy player coordinates
- Copy targeted block coordinates
