# CoordsCopy

A fabricmc minecraft mod which enabled you to copy player coordinates and block coordinates with the press of a button.

The mod provides you with a function to copy the coordinates of the block you're looking at directly into the
clipboard by pressing one simple button. The coordinates will be formatted in the common format also used in commands, which
is "x y z", (for example: -34 65 219).
The mod is also able to copy your current player position into the clipboard using another key, formatted the same as the
block coordinates.

### Current Features (v1.0):
- Copy player coordinates
- Copy targeted block coordinates
